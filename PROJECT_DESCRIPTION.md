# Branded Analytical Dashboard — Описание проекта

Этот проект представляет собой современное веб-приложение для создания интерактивных аналитических дашбордов на основе загружаемых данных (Excel) с автоматическим применением корпоративного брендинга.

---

## 🎨 Основная концепция
Пользователь загружает логотип компании и файл с данными. Система автоматически извлекает основные цвета из логотипа и применяет их ко всем элементам интерфейса: кнопкам, графикам, индикаторам и акцентам. Это позволяет мгновенно получить профессионально выглядящий отчет, готовый к презентации.

---

## ✨ Ключевые возможности

### 1. Интеллектуальная работа с данными
- **Обработка на клиенте**: Использование библиотеки `Arquero` (аналог Pandas для JS) позволяет быстро фильтровать и агрегировать миллионы строк прямо в браузере.
- **Автоматическое определение метрик**: Система сама находит числовые показатели (KPI) и измерения (категории) в загруженных таблицах.
- **Поддержка Excel**: Прямой импорт данных с помощью `XLSX`.

### 2. Динамический брендинг
- **Экстракция цветов**: При загрузке логотипа используется `ColorThief` для определения доминирующих цветов.
- **Глобальная стилизация**: Цвета автоматически обновляют CSS-переменные, адаптируя весь дашборд под стиль бренда.

### 3. Продвинутая аналитика
- **Интерактивная фильтрация**: Кросс-фильтры по категориям (клиенты, товары, города) и гибкий выбор периодов (годы, кварталы, месяцы).
- **KPI-панели**: Ключевые показатели эффективности с расчетом динамики.
- **Визуализация Tremor**: Использование профессиональных библиотек для построения графиков (Area Charts, Bar Charts, Donut Charts).

### 4. Управление отчетами (Report Library)
- Возможность сохранять несколько конфигураций отчетов.
- Быстрое переключение между разными наборами данных и настройками брендинга.
- Постоянное хранение метаданных в `LocalStorage`.

## 🛠 Технологический стек

- **Core**: React 18 + TypeScript + Vite.
- **State Management**: **Zustand** (быстрое и легкое состояние).
- **Data Engine**: **Arquero** (мощная манипуляция данными).
- **Styling**: **Tailwind CSS** + Radix UI.
- **Charts**: **Tremor** (UI-фреймворк для дашбордов).
- **Animations**: **Framer Motion** (плавные переходы и микровзаимодействия).
- **Icons**: Lucide React.

---

## 📂 Структура проекта

- `src/components`: Основные блоки интерфейса (Sidebar, Header, Main Grid).
- `src/components/ui`: Переиспользуемые атомарные компоненты (кнопки, диалоги, карты).
- `src/store`: Глобальное состояние приложения на Zustand.
- `src/lib`: Утилиты для обработки данных, даты и работы с цветом.
- `src/index.css`: Глобальные стили и конфигурация темы.

## 🚀 Как запустить

1.  Установите зависимости:
    ```bash
    npm install
    ```
2.  Запустите сервер для разработки:
    ```bash
    npm run dev
    ```
3.  Для сборки проекта:
    ```bash
    npm run build
    ```
